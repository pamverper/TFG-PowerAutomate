# Automatización del proceso de gestión de TFG con RPA

Este repositorio contiene los archivos exportados de los flujos de automatización desarrollados para el Trabajo Fin de Grado **“Automatización de procesos con RPA”**.

El objetivo del proyecto es implementar un prototipo de apoyo a la gestión administrativa de Trabajos Fin de Grado mediante herramientas de bajo código y automatización. La solución se apoya principalmente en **Power Automate**, **Power Apps**, **SharePoint** y **Make.com**, permitiendo reducir tareas manuales, centralizar información, mejorar la trazabilidad de los expedientes y automatizar comunicaciones asociadas al proceso.

## Tecnologías utilizadas

- **Power Automate**: implementación de los flujos de automatización.
- **SharePoint**: repositorio de datos del prototipo.
- **Power Apps**: interfaz de usuario para la gestión de expedientes.
- **Make.com**: integración complementaria con servicios externos e inteligencia artificial.
- **OpenAI**: apoyo en la resolución automática de consultas en determinados escenarios.
